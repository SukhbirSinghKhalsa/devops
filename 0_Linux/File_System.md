# Linux File System Structure

| Directory | Description |
|------------|--------------------------------------------|
| `/`      | Root directory (top-level of filesystem) |
| `/bin`   | Essential user command binaries |
| `/boot`  | Bootloader and kernel files |
| `/dev`   | Device files |
| `/etc`   | System-wide configuration files |
| `/home`  | User home directories |
| `/lib`   | Essential shared libraries |
| `/media` | Removable media mount points |
| `/mnt`   | Temporary mounted filesystems |
| `/opt`   | Optional / third-party software packages |
| `/proc`  | Virtual filesystem for process & kernel info |
| `/root`  | Root user's home directory |
| `/run`   | Runtime process data |
| `/sbin`  | System administration binaries |
| `/srv`   | Data served by system services |
| `/sys`   | Kernel and hardware information |
| `/tmp`   | Temporary files |
| `/usr`   | User applications and system resources |
| `/var`   | Variable data (logs, mail, spool, etc.) |
