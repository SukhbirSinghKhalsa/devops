# My Understanding Growing Each Day on Kubernetes

## keys understanding
```bash
- apiVersion = <GROUP> /VERSION  --> group present use this
             = VERSION --> when group not present
- GV = apiVersion [ GROUP VERSION]
- K  = Kind

- metadata - name --> pod name

- metadata - labels --> features of pod

- kind              --> what kind of services is used -- Pod, Deployment, NetworkPolicy, Service
```
